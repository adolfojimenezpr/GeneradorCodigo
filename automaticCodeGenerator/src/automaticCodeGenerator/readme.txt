Para ejecutar el programa es necesario importar el proyecto en Eclipse debido a que al intentar ejecutar el 
proyecto por consola se produce un error NULL POINTER EXCEPTION que al ejecutarlo con Eclipse no se produce. 
Adjuntamos el ejecutable como se pide en los requisitos de la práctica aunque no es posible ejecutarlo de esta 
manera por el problema antes descrito. Para ejecutar el proyecto hay que ejecutar la clase principal “Interfaz”.
